## 기능 설명

초기 원금·연 이자율·복리 주기·투자 기간·월 추가 납입액을 입력하면 **최종 자산, 총 이자, 수익률**을 즉시 계산합니다.  
연도별 누적 성장 차트와 상세 내역 테이블로 복리의 힘을 직관적으로 확인할 수 있습니다.

---

## 주요 기능

- **복리 계산** — 월/분기/반기/연 단위 복리 선택 지원
- **월 추가 납입** — 정기 적립식 투자 시뮬레이션 가능
- **핵심 지표** — 최종 금액 / 총 이자 / 수익률 / CAGR / 원금 2배 도달 기간
- **성장 차트** — 납입 원금 vs 누적 이자 누적 막대 그래프
- **연도별 테이블** — 각 연도의 납입 원금·누적 이자·최종 금액·수익률 확인

---

## 사용 방법

1. **초기 원금** 입력 (예: 1,000만 원)
2. **연 이자율** 입력 (예: 7%)
3. **복리 주기** 선택 (기본값: 월복리)
4. **투자 기간** 입력 (예: 10년)
5. **월 추가 납입** 입력 (없으면 0 또는 비워두기)
6. **계산하기** 버튼 클릭 (또는 Enter)

---

## 계산 공식

| 항목 | 공식 |
|------|------|
| 원금 복리 성장 | A = P × (1 + r/n)^(n×t) |
| 월 납입 미래가치 | FV = PMT × ((1 + r/12)^m − 1) / (r/12) × (1 + r/12) |
| 최종 금액 | 원금 복리 성장 + 월 납입 미래가치 |
| 수익률 (ROI) | (최종 금액 − 총 납입 원금) ÷ 총 납입 원금 × 100 |
| CAGR | (최종 금액 ÷ 초기 원금)^(1/기간) − 1 |
| 2배 도달 기간 | ln(2) ÷ ln(1 + 연이율) |
