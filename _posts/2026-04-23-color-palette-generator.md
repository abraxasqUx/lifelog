## 기능 설명

기준 색상을 입력하면 색상 이론에 기반한 **6가지 조화 유형**의 팔레트를 즉시 생성합니다.  
각 색상의 HEX / RGB / HSL 값을 클릭 한 번으로 복사할 수 있으며, 전체 팔레트를 CSS 변수 형식으로 내보낼 수 있습니다.

---

## 주요 기능

- **색상 입력** — 네이티브 컬러 피커 + HEX 직접 입력 (실시간 동기화)
- **랜덤 생성** — 버튼 클릭으로 새로운 기준 색상 무작위 생성
- **6가지 조화 유형** — 단색 / 보색 / 유사색 / 삼각형 / 사각형 / 분열 보색
- **값 복사** — 스와치 클릭 시 HEX 복사, 각 행의 복사 버튼으로 HEX·RGB·HSL 개별 복사
- **CSS 내보내기** — 팔레트 전체를 `:root { --color-1: ...; }` 형식의 CSS 변수로 복사

---

## 사용 방법

1. 상단 컬러 피커 또는 HEX 입력란에서 기준 색상을 선택
2. 원하는 색상 조화 유형 버튼 클릭
3. 생성된 스와치 클릭 → HEX 값 복사, 또는 각 행의 **복사** 버튼으로 원하는 형식 선택
4. **CSS 변수로 복사** 버튼으로 전체 팔레트를 CSS에 바로 붙여넣기

---

## 색상 조화 유형 설명

| 유형 | 설명 | 스와치 수 |
|------|------|-----------|
| 단색 (Monochromatic) | 동일 색조, 명도 5단계 변화 | 5 |
| 보색 (Complementary) | 기준색 + 색상환 반대편(+180°), 각 3단계 | 6 |
| 유사색 (Analogous) | 기준색 ±30°, ±60° 인접 색상 | 5 |
| 삼각형 (Triadic) | 색상환을 120° 간격으로 3등분 | 3 |
| 사각형 (Tetradic) | 색상환을 90° 간격으로 4등분 | 4 |
| 분열 보색 (Split-Complementary) | 보색 양옆 ±30° 두 색상 + 기준 3단계 | 5 |
