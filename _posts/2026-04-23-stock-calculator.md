## 기능 설명

매수가·수량·매도가를 입력하면 수수료와 증권거래세를 반영한 **실제 순손익과 수익률**을 계산해줍니다.  
손익분기가(Break-even Price)도 함께 표시되며, 매도가 변화에 따른 수익률 막대 차트도 시각화합니다.

---

## 주요 기능

- **실제 수익률 계산** — 매수·매도 수수료, 증권거래세를 모두 반영한 순손익
- **손익분기가** — 수수료·세금을 고려한 정확한 손익분기 매도가 자동 산출
- **수익률 차트** — 현재 매수가 기준 −40% ~ +80% 범위의 예상 수익률 시각화
- **세금 유형 선택** — KOSPI(0.18%) / KOSDAQ(0.23%) / 없음 / 직접 입력

---

## 사용 방법

1. **매수가**, **매수 수량**, **매도가** 입력
2. 수수료율 확인 (기본값: 0.015%)
3. 증권거래세 유형 선택
4. **계산하기** 버튼 클릭

---

## 계산 공식

| 항목 | 공식 |
|------|------|
| 실제 투자비용 | 매수금액 + 매수수수료 |
| 실제 수령금액 | 매도금액 − 매도수수료 − 증권거래세 |
| 순손익 | 실제 수령금액 − 실제 투자비용 |
| 수익률 | 순손익 ÷ 실제 투자비용 × 100 |
| 손익분기가 | 실제 투자비용 ÷ (수량 × (1 − 매도수수료율 − 세율)) |
